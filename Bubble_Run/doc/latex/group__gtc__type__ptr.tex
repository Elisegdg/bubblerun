\doxysubsection{Detailed Description}
Handles the interaction between pointers and vector, matrix types. 

This extension defines an overloaded function, \mbox{\hyperlink{group__gtc__type__ptr_gaa6abca1c01d37b11c0d524bdf74b2094}{glm\+::value\+\_\+ptr}}, which takes any of the \mbox{\hyperlink{group__core__template}{core template types}}. It returns a pointer to the memory layout of the object. Matrix types store their values in column-\/major order.

This is useful for uploading data to matrices or copying data to buffer objects.

Example\+:

\DoxyCodeLine{\textcolor{preprocessor}{\#include <\mbox{\hyperlink{third-party_2include_2glm_2glm_8hpp}{glm/glm.hpp}}>}}
\DoxyCodeLine{\textcolor{preprocessor}{\#include <\mbox{\hyperlink{type__ptr_8hpp}{glm/gtc/type\_ptr.hpp}}>}}
\DoxyCodeLine{}
\DoxyCodeLine{\mbox{\hyperlink{structglm_1_1detail_1_1tvec3}{glm::vec3}} aVector(3);}
\DoxyCodeLine{\mbox{\hyperlink{structglm_1_1detail_1_1tmat4x4}{glm::mat4}} someMatrix(1.0);}
\DoxyCodeLine{}
\DoxyCodeLine{glUniform3fv(uniformLoc, 1, \mbox{\hyperlink{group__gtc__type__ptr_gaa6abca1c01d37b11c0d524bdf74b2094}{glm::value\_ptr}}(aVector));}
\DoxyCodeLine{\mbox{\hyperlink{dummy_8cpp_a0294e2a1faa1a1ff720a35ac06b4282a}{glUniformMatrix4fv}}(uniformMatrixLoc, 1, \mbox{\hyperlink{dummy_8cpp_abca6d43f43fae31f49dcb883b2f301f6}{GL\_FALSE}}, \mbox{\hyperlink{group__gtc__type__ptr_gaa6abca1c01d37b11c0d524bdf74b2094}{glm::value\_ptr}}(someMatrix));}
