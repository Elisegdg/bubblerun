\doxysubsection{Detailed Description}
Functions and types that the GLSL specification doesn\textquotesingle{}t define, but useful to have for a C++ program. 

GTC extensions aim to be stable.

Even if it\textquotesingle{}s highly unrecommended, it\textquotesingle{}s possible to include all the extensions at once by including \texorpdfstring{$<$}{<}\mbox{\hyperlink{ext_8hpp}{glm/ext.\+hpp}}\texorpdfstring{$>$}{>}. Otherwise, each extension needs to be included a specific file.
